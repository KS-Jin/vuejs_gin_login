# vuejs_gin_login
使用 `Vuejs` 與 `Golang` 建立的登入系統  
實現 `前後端分離` `跨網域存取` `Json Token Web`  

#### 使用 VueJS 之插件
    1. axios
    2. qs
    
#### 使用 Golang 之插件
    1. gin
    2. jwt-go
    3. cors
    4. mgo
